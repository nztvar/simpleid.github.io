Simpleweb is a minimalist web framework.  It is similar to {@link http://webpy.org web.py}, but in PHP. The key to Simpleweb is the <i>route array</i>.  The route array is an array that maps URLs (called <i>patterns</i>) to PHP functions or methods (called <i>routes</i>).
